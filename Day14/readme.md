## 코드 작성 방법
1. Breakedown the Problem : 문제를 생각하고 작게 쪼개기
   - 알고리즘으로 그리면 더 좋다.
2. Make a Todo list and Start with the easiest : 투두 리스트를 만들고 가장 시작하기 쉬운 것 먼저 해결
3. Turn the problem into comments : 코드에 주석을 입력해서 설명
4. write code → run code → fix code
