# docstring
docstring을 사용하면 코드를 문서화하는 여러 줄 주석을 작성할 수 있음

```
"""
여기에 작성
"""
```

## 함수 문서화(Documenting Functions)
1. 함수 정의 바로 아래에 사용할 수 있음
2. 함수 호출 위에 마우스를 올리면 해당 텍스트가 표시됨

    ** 자체생성 함수가 무엇인지 확인할 수 있음

### 예시
```
def my_function(num):
    """Multiplies a number by itself."""
    return num * num
```
